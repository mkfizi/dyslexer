About

Dyslexer is an online tool font converter that convert fonts to dyslexic friendly font. This tool aims to increase accessibility and ease readibility for users who suffers dyslexia.

Dyslexer uses Open Dyslexic, a typeface specifically designed against some common symptoms of dyslexia. This tool is open sourced and can be used in any project.
